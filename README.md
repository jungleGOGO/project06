## 🗂 회원 기능
### 1. 로그인
- 이메일과 비밀번호 값으로 검증
- Spring Security 통한 로그인 인증 처리 구현
- 로그아웃
  <br>
  <br><br>

### 2. 회원가입
- valid 어노테이션 및 bindging result 활용하여 유효성 검사 진행
- 이메일 중복처리 구현
- 작성한 이메일로 인증번호 발급
  <br>
  <br><br>

### 3. 비밀번호 찾기
- 가입한 이메일로 임시 비밀번호 발급
- 발급된 비밀번호 암호화 되어 DB에 저장
